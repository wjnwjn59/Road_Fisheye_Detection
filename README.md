# Road Fisheye Detection
## Description
Source code for AI CITY 2024 Challenge Track 4: Road Object Detection in Fish-Eye Cameras.

## Datasets
Download Fisheye8K dataset at [here](https://scidm.nchc.org.tw/en/dataset/fisheye8k).

## Methods

- [x] YOLOv8
- [] Co-DETR
